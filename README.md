## 功能

| 功能 | 描述 |
|------|------|
| 标签编辑 | 标题、艺术家、专辑、封面 |
| 视频提取 | 从视频中提取音轨 |
| 敏感词检测 | 检查元数据敏感词 |
| 批量操作 | 批量填充、保存 |
